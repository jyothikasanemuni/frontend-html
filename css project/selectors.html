Selectors

A selector is used in CSS to select and style HTML elements.It tells the browser which elements the CSS rules should apply to.

Types of Selectors

  1. Basic Selectors
  2. Combinator (Combination) Selectors
  3. Pseudo Selectors

1) BASIC SELECTORS

a) Tag (Element) Selector
Selects all elements by tag name.

Syntax:
      tagname {
      property:value;
      }

Example:
      p{
      color:blue;
      }

b) Class Selector(.)
Selects elements with a specific class name (use a dot .)

Syntax:
      .classname {
        property:color;
      }

Example:
      .text{
      color:blue;
      }

c) ID Selector(#)
Selects a single element with a specific ID (use a hash #).
Syntax:
      #idname {
      property:value;
      }

Example:
      #text{
      color:blue;
      }

d) Group Selector(,)
Groups multiple selectors to apply the same styles.
Syntax:
      selector1,selector2 {
      propert:value;
      }

Example:
      h1,h2,p{
      color:blue;
      }

e) Universal Selector(*)
Selects all elements in the HTML document.
Syntax:
      * {
      property:value:
      }

Example:
      * {
      color:blue;
      }

2) Combinator (Combination) Selectors

a) Descendant Selector (Space)
Selects all direct and indirect children for styling.
Syntax:
      parent child {
      property:value;
      }

Example:
      .parent p{
      background-color:blue;
      }

b) Child Selector ( > )
Selects direct child elements of a parent.
Syntax:
      parent>child {
      property:value;
      }

Example:
      .parent p{
      background-color:blue;
      }

c) Adjacent Sibling Selector ( + )
Selects an element that is immediately next to another element.
Syntax:
      element1+element2 {
      property:value;
      }

Example:
      h1 + p {
      background-color:blue;
      }

d) General Sibling Selector ( ~ )
Selects all elements that are siblings of a specified element.
Syntax:
      element1 ~ element2 {
      property:value;
      }

Example:
      h1 ~ p {
      background-color:blue;
      }

3)Pseudo Selectors

There are two types:

  • Pseudo-class selectors(:)
  • pseudo-element selectors(::)

A) Pseudo-Class Selectors (:)

:Hover
styles element mouse hovers over an element

Example:
      button:hover {
      background-color:green;
      }

:link
A pseudo-class that selects all unvisited links

Example:
      a:link {
      color:red;
      }

:active
styles element when it is being clicked

Example:
      a:active {
      color:red;
      }

:visited
Styles links after they are visited

Example:
      a:visited {
      color:purple;
      }

:focus
Styles input when focused (clicked/tabbed)

Example:
      input:focus {
      border: 2px solid blue;
      }

:checked
Selects checked inputs

Example:
      input:checked {
      background-color:yellow;
      }

:first-child
Selects the first child element inside a parent.

Example:
      p:first-child {
      color:violet;
      }

:last-child
Selects the last child element inside a parent.

Example:
      p:last-child {
      color:lightgreen;
      }

:nth-child(n)
Selects nth child.

Example:
      p:nth-child(3) {
      color:blue;
      }

:nth-of-type(n)
A pseudo-class that selects the nth element of its type (like the 2nd <p>, 3rd <li>, etc..) inside its parent.

Example:
      p:nth-of-type(2) {
      color:orange;
      }

B) Pseudo-Element selectors(::)

::first-letter
Styles the first letter of a block element.

Example:
      p::first-letter {
      font-size: 30px;
      }

::first-line
Styles the first line of a block element.

Example:
      p::first-line {
      color: green;
      }

::before

Example:

::after
Inserts generated content after an element.

Example:

::selection
Styles the text when it is highlighted (selected)

Example:
      ::selection {
      background-color:yellow; color: black;
      }
