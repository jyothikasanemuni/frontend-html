TYPOGRAPHY

Typography in CSS refers to the style, appearance, and arrangement of text on a webpage.It helps make content readable, attractive, and well-structured using different font and text properties.

TEXT PROPERTIES:-

Text properties are css styles used to control the apperance, alignment, spacing & decoration of the text on a webpage.

Types of text properties:

1) Text color:
Defines the text color.
Example:
          color:black;

Lorem ipsum dolor sit amet.

2) Text alignment:
Controls how text is aligned inside its container.
values:
          text-align:center;
          text-align:left; (default)
          text-align:right;

Examples:

lorem

lorem

lorem

3) Text decoration:
Adds or remove decorations from text.
values:
          text-decoration:underline;
          text-decoration:overline;
          text-decoration:line-through;
          text-decoration:underline dotted;
          text-decoration:none;

Examples:

jyothika

prasanna

moulika

bailey

Anu

4) Text transform:
Changes the capitalization of text.
values:
          text-transform:uppercase;
          text-transform:lowercase;
          text-transform:capitalize;

Examples:

jyothika

Moulika

jyothika sanemuni

5) Text indent:
Changes the capitalization of text.
values:
          text-indent: 90px;

Example:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Consequuntur, exercitationem?

6) Text-Shadow:
Adds shadow to text.
Syntax: text-shadow: h-offset v-offset blur-radius color;

Example:

JYOTHIKA SANEMUNI

7) Line-Height:
Controls vertcal spacing between lines of text.
Syntax: line-height: 70px;

Example:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Amet sed voluptates possimus temporibus repellendus nobis, nihil eveniet voluptatibus sequi beatae harum? Unde, fugit dolores sint quaerat iusto ipsa delectus ducimus esse et quidem nemo enim, officiis vitae? Cum est perferendis aperiam, quia libero suscipit alias, possimus id molestias ducimus veritatis!

8) Letter-Spacing:
Adjusts space between characters.
Syntax: line-spacing: 5px;

Example:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Aliquid, harum.

9) Word-Spacing:
Adjust space between words.
Syntax: word-spacing: 20px;

Example:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Fugit, quasi!
