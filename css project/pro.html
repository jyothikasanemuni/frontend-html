CSS

CSS stands for Cascading Style Sheets.It is used to style and design web pages.CSS describes how HTML elements should be displayed on the screen such as their color, font, layout, spacing, and background.It helps make web pages attractive, consistent, and easy to maintain.

  • To apply same design on multiple pages easily.
  • To separate content (HTML) from design (CSS).
  • To improve website performance and maintenance.

SELECTORS

A selector in CSS is used to select the HTML elements you want to style.It tells the browser which elements the CSS rules should apply to.

BOX MODELüéÅ

It consists of four parts:
Content:- The actual text or image inside the box.
Padding:- Space between content and border.
Border:- The line surrounding the padding.
Margin:- Space outside the border (distance from other elements).
Together, these control the size and spacing of elements on a webpage.

COLOR THEORY

Color Theory is the concept of how colors work together to create a visually pleasing design.
Colors are categorized into Primary (red, blue, yellow), Secondary, and Tertiary.CSS uses colors through names, HEX codes, RGB, RGBA, HSL,HSLA etc.Good color choice improves readability, mood, and user experience on a website.

TYPOGRAPHY

Typography is the art and technique of styling and arranging text on a webpage.It makes content readable, attractive, and well-organized.In CSS, typography is controlled using font and text properties.
Common properties: font-family, font-size, font-weight, color, line-height, text-align, text-transform.Good typography improves user experience and visual appeal of a website.
