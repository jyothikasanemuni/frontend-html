BOX MODEL

The CSS Box Model describes how every HTML element is treated as a rectangular box. It defines how content, padding, border, and margin work together to determine the space an element takes up on a webpage.

It consists of four parts:

  • Content
  • Padding
  • Border
  • Margin

CONTENT

The actual area where text, images, or other elements appear.
Controlled using properties like width and height.

Example:
          p{
          width:200px;
          height:300px;
          }

PADDING

The space between the content aIt can be set for each side: padding-top, padding-right, padding-bottom, padding-left.

Example:
          p{
          padding-top:20px;
          padding-left:30px;
          padding-bottom:15px;
          padding-right:27px;
          }

BORDER

The line surrounding the padding and content.
Controlled using properties like border-width, border-style, and border-color

Example:
          p{
          border: 2px solid black;
          border: 2px dotted green;
          border: 2px double pink;
          border: 2px groove blue;
          border: 2px inset brown;
          border: 2px outset orange;
          border: 2px ridge red;
          border: 2px none violet;
          }

MARGIN

The space outside the border, creating distance between elements.
It can be set for each side: margin-top, margin-right, margin-bottom, margin-left.

Example:
          p{
          margin:2px;     -->applies to all sides..
          margin-top:20px;
          margin-left:30px;
          margin-bottom:15px;
          margin-right:27px;
          margin:auto;    -->brings the box horizaontally into center.          
          }
