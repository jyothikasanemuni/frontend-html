It's one of the frontend technology. CSS is used to styling a webpage , not only styling we can align the contet and arrange the html elements.CSS stands for Cascading Style Sheets

The word "Cascading" means order of priority when multiple styles are applied to the same element.

1)INLINE CSS

CSS is written inside the HTML tag using the style attribute.
Syntax:<h1 style="color:blue;">Inline CSS<\h1>

Advantage:Easy and quick to apply for single elements.
Disadvantage: Not reusable and hard to maintain for large websites.

2)INTERNAL CSS

We can write the css code inside of style tag it should be placed inside the head tag.

Syntax:
<head>
     <style>
          h1{
          Color:white;
          background-color:red;
          }
     </style>
</head>

Advantage: Styles multiple elements in the same page
Disadvantage: Works only for that single HTML page (not reusable across many pages)

3)EXTERNAL CSS

CSS is written in a separate file (with .css extension) and linked using the <link> tag.

Syntax: <link rel="stylesheet" href="./style.css">

Advantage: External CSS keeps code clean,reusable, consistent across pages, and improves performanvce with caching.
Disadvantage: Requires exttra file load, can delay rendering and if the file is missing the page looks unstyled.
