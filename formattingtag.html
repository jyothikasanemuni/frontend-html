Text Formatting tags:

Formatting tags are uscd to change the appearance of text like making it bold, italic, underlined, small, big, marked etc...

<b>(Bold Text):

Makes the text bold without giving it extra importance.

<strong>Strong (Strong Importance):

Highlights text as important; browsers render it in bold.

<i>(italic Text):

The text italic without empbasizing importance (used for style, like book titles).

<em>(Emphasized Test):

Emphasizes text; browsers render it in italic and screen readers put stress on it.

<u>(Underline):

Underlines the text.

<mark>(Highlighted Text):

Highlights text with a yellow background (like a marker).

<small>(Smaller Text):

Displays text one size smaller than normal.

<big>(big tag):

The <big> tag is tused to increase the font size of text by one level larger than the surrounding text. It was used in HTML4, but it is not supported in HTML.S (deprecated). Instead of <big>, we now use CSS for font sizing (font-size).

<del>(Deleted Text):

Shows text that has been deleted/removed, browsers display it with a strike-through.

<ins>(inserted Text):

Shows text that has been added/inserted; browsers display it as underlined.

<sub>(Subscript):

Displays text below the baseline (used in formulas, chemical notations).

<sup>(Superscript):

Displays text abeve the baseline (uscd in exponents, math notations).
