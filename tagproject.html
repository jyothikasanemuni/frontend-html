TAG

Tags are predefined words which helps to build the structure of a webpage. Tags are written by using angular barces <> .Tags have opening and closing tag.

Types of Tags

1) Paired tag/Container tag

These have both opening and closing tags.

2) Non-paired tag/Empty tag/Self closing tag

These don't need a closing tag

example: hr,br

Heading Tags

Heading tags are used to define titles or headings in a webpage.There are 6 levels of headings where h1 is the largest, h6 is the smallest.

Example:

Heading 1

Heading 2

Heading 3

Heading 6

Paragraph Tag

This tag defines a paragraph of text..

Example:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Ut, animi. Eius iusto mollitia modi, molestiae consequuntur fuga optio, voluptates sed dolor consectetur quaerat et, eveniet facere cum. Consequatur, cupiditate dolores.

PRE Tag

The tag displays text exactly as it is written in the HTML file including spaces, tabs, and line breaks.

example:

Name: Jyothika
Course: Full Stack Python
Duration: 6 Months
